# Сайт расчёта дохода, накоплений и активов с учётом инфляции

MVP-проект на актуальном стеке Next.js 16 + React 19 + TypeScript. Реализует расчёт фактических и идеальных значений по годам, работу со встроенными и загружаемыми инфляционными наборами, графики, итоговые карточки и серверное сохранение расчётов в JSON.

## Что реализовано

- анонимный режим в у.е.;
- режим `у.е. + валюта` с курсом `1 у.е. = X валюты`;
- встроенные наборы инфляции для России, США, Еврозоны, Казахстана и Беларуси;
- загрузка инфляции из CSV и JSON;
- расчёт идеальных значений по инфляции;
- расчёт фактических значений по логике:
  - линейный рост;
  - ступенчатый рост;
  - промежуточные точки;
- расчёт суммарного дохода за период для месячного и годового режима;
- таблица по годам;
- графики:
  - столбчатая диаграмма дельты дохода;
  - линейные графики фактическое vs идеальное для дохода, накоплений и активов;
- сохранение расчёта на сервере в JSON;
- `anonymous_user_id` через cookie + localStorage;
- сохранение последней формы в localStorage.

## Установка через `install.sh`

Для быстрого развёртывания на Ubuntu/WSL можно использовать скрипт `install.sh`.  
Он устанавливает Node.js 22.x из официального репозитория NodeSource, очищает старые артефакты (`node_modules`, `.next`, `package-lock.json`), устанавливает зависимости и собирает проект.

Пример использования (изнутри директории проекта):

```bash
chmod +x ./install.sh
./install.sh "$(pwd)"
```

Если нужно указать другой путь к директории проекта:

```bash
./install.sh /home/username/inflation-income-site
```

Скрипт требует прав `sudo` для установки системных пакетов и репозитория NodeSource.

## API

### Получить список стран

```bash
GET /api/inflation/countries
```

### Получить инфляцию по стране

```bash
GET /api/inflation/RU
GET /api/inflation/US
```

### Сохранить расчёт

```bash
POST /api/calculation/save
Content-Type: application/json
```

## Формат пользовательского файла инфляции

### CSV

```csv
year,inflation_percent
2023,7.4
2024,6.1
2025,5.8
```

### JSON

```json
[
  { "year": 2023, "inflation_percent": 7.4 },
  { "year": 2024, "inflation_percent": 6.1 },
  { "year": 2025, "inflation_percent": 5.8 }
]
```

## Где сохраняются JSON-файлы

Файлы сохраняются в директорию:

```bash
data/calculations/<anonymous_user_id>/
```

Каждый расчёт получает отдельный JSON-файл.
